# RQ1

This repository contains the analysis scripts as used in the study.

The code is provided for transparency and reproducibility and has not
been refactored or generalised. Scripts may require adaptation for
different computing environments.

Data and Models are not included and must be obtained from MARRMoT (Knoben et al., 2019; Trotter et al., 2022) and CAMELS-AUS v2
(Fowler et al., 2025).
